## ep-classification

Репозиторий содержит модульный конвейер для классификации эпох (промежутков фиксированной длины — многомерных сигналов ЭЭГ) по наличию / отсутствию намерения человека совершить самоиницированное движение при помощи метрических методов.

### Установка
1. Установите Miniconda/Anaconda.
2. Создайте и активируйте окружение:
```bash
conda env create -f env.yml
conda activate my_env
```

### Запуск
- Запуск основного пайплайна:
```bash
python pipelines/exp_main.py
```
- Запуск пайплайна с автоматическим вычислением важностей признаков:
```bash
python pipelines/exp_main_importances.py
```

### Структура проекта
- `pipelines/` — сценарии запуска экспериментов
- `configs/` — файлы с параметрами экспериментов
- `src/` — реализация абстрактных компонентов (один файл = один абстрактный класс; внутри — разные реализации):
  - `__init__.py`
  - `averager.py` — класс Averager
  - `comparator.py` — класс CandidatesComparator
  - `distance_calculator.py` — класс DistanceCalculator
  - `selector.py` — класс CandidateSelector
  - `smoother.py` — класс Smoother
  - `trainer.py` — класс Trainer
  - `utils.py` — вспомогательные функции
- `notebooks/` — jupyter-notebook'и с визуализациями 

### Общая схема алгоритмов
1. Определение референтов в имеющемся наборе эпох
2. Подсчёт расстояний до референтов и фильтрация по расстоянию  
3. Выбор класса эпохи на основе классов референтов финального списка и расстояний до них

Опциональный предварительный шаг — сглаживание отдельных сигналов или эпох целиком.

### Описание классов
- **DistanceCalculator**  
  Класс для вычисления расстояния между эпохами. Поддерживает базовые метрики (косинусное сходство, евклидово расстояние, корреляцию и др.), алгоритм DTW и любые функции, принимающие два объекта и возвращающие значение, количественно характеризующее ощущение расстояния между ними.

- **Smoother**  
  Класс для сглаживания сигналов по выбранным каналам методом скользящего среднего. Принимает размер окна как параметр.

- **Averager**  
  Класс для «усреднения» эпох в широком смысле. По набору эпох и объекту DistanceCalculator возвращает список референтов — датафреймов, относительно которых строится прогноз для классифицируемой эпохи.

- **Trainer**  
  Класс, реализующий логику обучения модели. Параметры: тренировочные эпохи с их лейблами (позитивная/негативная), референты, объект DistanceCalculator.

- **CandidateSelector**  
  Класс, осуществляющий подсчёт расстояний до референтов и формирование финального списка (например, через выбор k ближайших).

- **CandidatesComparator**  
  Класс, присваивающий эпохе лейбл по расстояниям до финального списка референтов.
